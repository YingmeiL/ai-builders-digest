Dario Amodei 公开数据: Anthropic 80 倍增长 Anthropic CEO
🔥 重磅 行业里程碑 增长数据

Anthropic CEO Dario Amodei 在 Code with Claude 大会上首次公开透露：公司今年早些时候在使用量和营收上实现了 80 倍增长。他表示这一增速"疯狂到难以应对"，并强调将持续"获取一切能获取的算力"。

Claude Dreaming: Agent 学会"做梦"自我进化 Claude 官方
🔥 热门 全新功能 Agent 自主学习

Dreaming（做梦）功能正式发布：Claude 托管 Agent 可以在空闲时回顾过去的会话记录，提取行为模式，自主整理记忆，从而实现跨会话的持续学习和自我改进。

类似人类在睡眠中巩固记忆 — Agent 不再是"用完即忘"的工具，而是可以持续成长的智能体。目前作为研究预览版开放申请。

延伸：Ars Technica 报道称 Dreaming 此前已被泄露，正式发布说明 Anthropic 的 Agent 路线图执行力极强。WaveSpeed 的技术分析指出，Dreaming 可能是 Agent 从"工具"走向"同事"的关键转折点。

Claude Outcomes: Agent 输出质量可量化 Claude 官方
全新功能 质量控制 企业级

Outcomes 让你为 Agent 输出设置标准：你编写评分规则（rubric），一个独立的评估 Agent 检查输出质量，工作 Agent 会反复迭代直到达到你设定的标准。完成后通过 Webhook 通知。

这解决了 AI Agent 部署中最大的痛点之一："怎么保证输出质量一致且可靠"。对企业级 Agent 应用意义重大。

3. 前沿 AI 突破网络攻击"红线"

Nathan Benaich 的《State of AI 2026年5月刊》指出，前沿 AI 模型已跨越进攻性网络攻击能力的关键门槛。AI 不仅能防御，也开始具备发动复杂网络攻击的能力，引发安全界严重关切。

4. 英国 DSIT 警告 AI 驱动的网络威胁快速增长

英国科学创新技术部 (DSIT) 发布警告称，AI 赋能的网络攻击频率和复杂度正快速增加。企业需要将 AI 安全防御纳入核心基础设施规划。

5. OpenAI 发布新语音智能功能

OpenAI 推出新一代语音能力，进一步拓展多模态交互的边界。语音 AI 正从简单的语音转文字走向真正的语音智能交互，这将为 AI 助手的使用场景带来新的可能。
