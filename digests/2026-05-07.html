📝 官方博客更新

Claude 官方博客 Anthropic
全新功能 Agent 自我学习 多 Agent 协作

《Claude 托管 Agent 新功能：Dreaming、Outcomes 和多 Agent 编排》

今天发布的三大新功能：

1. Dreaming（做梦）：作为研究预览推出。Dreaming 让 Agent 在空闲时回顾过去的会话记录，发现行为模式和改进空间。类似于人类在睡眠中巩固记忆，Agent 通过"做梦"实现自我进化——无需人工调优。据 Ars Technica 报道，这是"Agent 从被动工具走向主动学习者"的关键一步。

2. Outcomes（结果评估）：新增标准化的任务结果评估框架，让开发者可以量化 Agent 的执行效果，建立反馈闭环。

3. 多 Agent 编排：支持多个 Agent 协同工作，由一个 Agent 分配和协调其他 Agent 的任务，适用于复杂的企业级工作流。

行业影响：路透社报道称，Dreaming 功能的发布与 SpaceX 算力合作同日公布，显示 Anthropic 在同时推进"更强的算力"和"更聪明的 Agent"双线战略。XDA Developers 指出，Dreaming 此前曾被泄露，如今正式发布说明 Anthropic 的 Agent 路线图正在稳步推进。
